# DCTimer
安卓系统魔方计时器

# 主要功能
・ 计时：2位或3位小数精度，计时范围0~596小时；支持WCA观察；可通过触屏、手动输入或连接Stackmat三种方式计时
・ 打乱：所有WCA官方的打乱，包括2阶、3阶、金字塔、SQ1随机状态打乱；提供官方打乱的打乱状态查看；其他非官方打乱和训练打乱，比如三阶LL训练、高阶对棱训练等
・ 统计：显示分组平均及去尾平均，提供3/5/12/50/100次滚动平均查看；可查看分组的时间分布直方图、折线图等
・ 其他：三阶cross/xcross计算、SQ1复形计算等；可设置背景颜色或图片，各种文字颜色等
